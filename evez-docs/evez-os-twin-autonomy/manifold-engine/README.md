# Manifold System Engine

Unified engine combining quantum, memory, automation, and scanning.

## Modules

| Module | Path | Description |
|--------|------|-------------|
| Quantum | `quantum/` | IBM Qiskit circuits, Bell/GHZ states |
| Memory | `storage/` | Persistent AI memory with search |
| Automation | `automation/` | Browser workflow library |
| Scanner | `scanner/` | Affiliate/marketplace scanner |

## Usage

```bash
# Run full engine
python engine.py

# Run individual modules
python quantum/ibm_demo.py
python storage/memory.py  
python automation/workflows.py
python scanner/affiliate_scanner.py
```

## Quick Start

1. **Quantum** - Run circuit demos (needs `pip install qiskit`)
2. **Memory** - Add/search memories
3. **Automation** - Run browser workflows
4. **Scanner** - Explore affiliate programs
