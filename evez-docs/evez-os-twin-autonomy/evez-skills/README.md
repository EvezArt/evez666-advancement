# EVEZ Skills

AI agent skills built from EVEZ-OS modules. Ready for ClawHub marketplace.

## Skills

| Skill | Description |
|-------|-------------|
| evez-revenue-engine | Revenue execution across Fiverr, X, Consulting |
| evez-self-compiler | Python module validation and compilation |
| evez-ledger | Immutable event log with cryptographic chain |
| evez-awareness-engine | Real-time awareness scanning and alerts |

## Usage

Clone and use any skill:
```bash
git clone https://github.com/EvezArt/evez-skills.git
cd evez-skills/<skill-name>
```

## Publish

```bash
clawhub login
clawhub publish ./<skill-name>/
```
