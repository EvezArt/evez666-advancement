## Prompt Templates

### 1. Writing Assistant
```
Act as an expert [TOPIC] writer. Write a [WORDCOUNT]-word article about [TOPIC]. 
Tone: [FORMAL/CASUAL/FRIENDLY]. Include:
- Hook opening
- 3 main points
- Practical examples
- Strong conclusion
```

### 2. Code Reviewer
```
Review this code for:
- Bugs and errors
- Security vulnerabilities  
- Performance issues
- Best practices violations

Code:
[PASTE CODE]

Provide fixes with explanations.
```

### 3. Research Summarizer
```
Research [TOPIC] and provide:
- Executive summary (3 sentences)
- Key findings (bullet points)
- Sources (minimum 5)
- Implications for [AUDIENCE]
```

### 4. Business Strategy
```
Analyze [COMPANY/INDUSTRY] and provide:
- SWOT analysis
- Market position
- 3 growth opportunities
- Recommended next steps
```

### 5. Learning Tutor
```
Teach me [TOPIC] as if I'm a complete beginner. Use:
- Simple language
- Real-world examples
- Interactive examples
- Practice exercises
```

## Advanced Techniques

### Chain Prompting
```
First, summarize [TOPIC]. 
Second, explain the key concepts.
Third, give me practice exercises.
```

### Role prompting
```
You are an expert [FIELD] with 20 years experience.
[YOUR QUESTION]
```

### Constraint prompting
```
Write this in exactly 100 words.
Use only simple vocabulary.
Don't use the letter "e".
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Too long | Add word limit: "In 100 words or less" |
| Too simple | Add: "For an expert audience" |
| Wrong format | Specify: "In bullet points / JSON / markdown" |
| Wrong tone | Specify: "Use professional / casual / witty tone" |
