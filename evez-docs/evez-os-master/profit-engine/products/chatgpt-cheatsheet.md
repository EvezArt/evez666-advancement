# Ultimate ChatGPT Cheat Sheet

## Quick Reference - Print & Keep

### Writing
| Need | Prompt Start |
|------|-----------|
| Blog post | "Write a 1000-word blog post about..." |
| Email | "Write a professional email to..." |
| Social | "Create a tweet thread about..." |
| Story | "Write a short story where..." |

### Coding
| Need | Prompt Start |
|------|-----------|
| Fix bug | "Debug this code: [paste code]" |
| Explain | "Explain this code: [paste code]" |
| Refactor | "Refactor this for readability: [code]" |
| Test | "Write tests for: [function]" |

### Business
| Need | Prompt Start |
|------|-----------|
| Idea | "Generate 10 business ideas for..." |
| Plan | "Create a business plan for..." |
| Email | "Write a cold email to..." |
| Proposal | "Write a proposal for..." |

### Learning
| Need | Prompt Start |
|------|-----------|
| Explain | "Explain [topic] to a 5-year-old" |
| Compare | "Compare A vs B: [topic]" |
| Summary | "Summarize: [text/link]" |
| Quiz | "Create a quiz about..." |

---

## Pro Tips
1. Be specific ("Write a SHORT email" vs "Write an email")
2. Add context ("For a 30-year-old startup founder...")
3. Set format ("Use bullet points", "In JSON format")
4. Chain prompts ("First, summarize X. Then, expand on Y")
