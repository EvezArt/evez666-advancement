# EVEZ - OpenClaw Skill

## Description
EVEZ autonomous system operations - spine, agent, memory, cognition, swarm, finance

## Commands

### spine
Event spine operations - append, verify, query
```
python3 $WORKSPACE/evez_assets/launcher.py spine
```

### agent
Autonomous agent decisions
```
python3 $WORKSPACE/evez_assets/launcher.py agent
```

### memory
Memory store - search, store
```
python3 $WORKSPACE/evez_assets/launcher.py memory
```

### cognition
FIRE cognition events
```
python3 $WORKSPACE/evez_assets/launcher.py cognition
```

### loop
Autonomous improvement loop
```
python3 $WORKSPACE/evez_assets/launcher.py loop
```

### swarm
Multi-agent swarm orchestration
```
python3 $WORKSPACE/evez_assets/launcher.py swarm
```

### finance
Trading and portfolio management
```
python3 $WORKSPACE/evez_assets/launcher.py finance
```

### api
Start HTTP API server (port 8765)
```
python3 $WORKSPACE/evez_assets/api_server.py
```

### full
Run full integrated system
```
python3 $WORKSPACE/evez_assets/launcher.py full
```
