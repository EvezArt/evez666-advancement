## 2. Nodes and Circuits

### Node Components
- **Signal Input / Output ports** - Claims enter and exit here
- **Local Ledger** - Temporary claim storage
- **Verification module** - Runs audit functions
- **Entanglement ports** - Connect to other nodes
- **Prebunk validator** - Checks against distortion patterns

### Circuit Connections
- **Horizontal links**: Node ↔ Node entanglement (metadata sync, audit requests, signal replication)
- **Vertical links**: Node ↔ Layer (signal propagation, verification check, prebunk cross-link)
- **Feedback loops**: Discrepancy triggers audit, then propagates corrections back through layers

### Mesh Architecture
- Fully connected entanglement between nodes (partial replication to reduce load)
- Redundant paths for resilience
- Audit and correction signals propagate across multiple routes simultaneously
