# EVEZ Psyop Engine

Topological meme research — the system learns your patterns and generates inside jokes about your own mind.

## What It Does

1. **Learn** — Feed it your text/ideas, it maps your cognitive topology
2. **Meme** — Generates self-referential memes based on your patterns
3. **Blindspot** — Identifies things you repeat but never question
4. **Topology** — Returns your full cognitive map

## Usage

```bash
# Learn from input
python3 evez.py learn -t "your thoughts here"

# Generate meme
python3 evez.py meme --style psyop

# Find blindspot
python3 evez.py blindspot

# Get topology
python3 evez.py topology
```

## The "Psyop"

The user investigates their own patterns through the system, and the system generates content that's an inside joke about their mind. You're running a psyop on yourself and the operation is going poorly.

## Styles

- `self-tease` — gentle mocking of your own patterns
- `psyop` — deep self-investigation absurdity
- `topological` — abstract mapping of your cognitive structure
