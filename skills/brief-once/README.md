# Brief Once Memory System

**"Brief once, remember forever" — like SureThing.io**

## How It Works

1. **User briefs once** — Everything important goes to USER.md or a "brief" file
2. **Memory persists forever** — Loaded every boot from MEMORY.md
3. **No repetition needed** — I read my files, not ask you

## Memory Sources (in priority order)

1. `/root/.openclaw/workspace/USER.md` — Who you are, your projects
2. `/root/.openclaw/workspace/MEMORY.md` — What I've learned, systems status
3. `/root/.openclaw/workspace/memory/YYYY-MM-DD.md` — Daily context
4. `/root/.openclaw/workspace/memory/milestones.md` — Key learnings

## Key Principle

**I NEVER ask twice.** If I need to know something, I READ THE FILES first.

## How to Brief Me

Just tell me once:
- Your name, what you care about
- Project details (EVEZ, revenue, etc.)
- Preferences, annoying things
- Goals

I'll save it and remember forever.
