# Revenue Tracker Skill

**Track all revenue circuits in real-time.**

## What It Does

- Monitors 7 revenue circuits: MONEY_SPIN, CUSTOMER_CLOSER, ALERT_EMPIRE, etc.
- Tracks earnings from /root/.openclaw/workspace/money/
- Reports total projected revenue
- Alerts on significant changes

## Usage

```
revenue-tracker status    # Show all circuit statuses
revenue-tracker total     # Show total projected revenue
revenue-tracker alert     # Create alert if revenue drops
```
