Build blueprint

Layer 1 — Phone shell

A mobile-first command center with compact typography, sticky navigation, gesture-friendly cards, and one-screen mission control.

Layer 2 — Worker organism

An AI pet with stateful energy, mood, growth, and work cycles. Success feeds evolution. Errors lower health and trigger self-heal routines.

Layer 3 — Sandbox internet

A browser-tab colony that simulates research, routing, tool use, and nested task execution so the app feels active rather than static.

Layer 4 — Successor forge

A visible mechanism where the system proposes stronger prompts, stronger skills, and stronger descendants that could outperform the current shell.
