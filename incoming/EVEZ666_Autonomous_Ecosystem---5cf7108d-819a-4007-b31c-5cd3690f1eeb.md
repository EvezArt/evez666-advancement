### Agent Descriptions

**Twitter Intelligence Scraper** (`bots/twitter_scraper.py`) scrapes @EVEZ666 tweets via Nitter RSS (no auth required) with snscrape and TwStalker API as fallbacks. It builds a living JSONL corpus with SHA-256 deduplication and extracts voice patterns including aphorisms, FIRE event posts, and topology posts. The resulting voice model is consumed by the meme generator.

**Messiah Meme Agent** (`agents/messiah_meme_agent.py`) generates memes in EVEZ666's voice using the trained persona corpus. It uses GPT-4.1-mini for LLM-powered caption generation with template fallback, renders text-based meme images via PIL, and logs every generated meme to the hash-chained meme spine. A constitutional guard prevents any harmful content from being generated.

**Self-Developer Agent** (`agents/self_developer.py`) audits all 23 EvezArt GitHub repositories for stub files, TODOs, and incomplete implementations. For HIGH and CRITICAL priority repos, it uses LLM to generate completions and logs every decision to the dev spine. It generates a comprehensive ecosystem status report on each cycle.

**FIRE Event Monitor** watches the `evez-os` EventSpine for new FIRE events. When an EXTREME_FIRE event is detected (tau >= 20), it triggers the Terra Incognita ARG realm and generates a special meme batch.

**Immortal Watchdog** monitors all agent threads and automatically restarts any that die, ensuring the system never goes offline.
